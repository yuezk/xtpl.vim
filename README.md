# xtpl.vim
XTemplate Syntax file for Vim

## 如何安装

使用 [Vundle][vundle]:

```
Plugin 'yuezk/xtpl.vim'
```

使用 [vim-plug][plug]:

```
Plug 'yuezk/xtpl.vim'
```

使用 [NeoBundle][bundle]:

```
NeoBundle 'yuezk/xtpl.vim'
```

[vundle]: https://github.com/gmarik/vundle
[plug]: https://github.com/junegunn/vim-plug
[bundle]: https://github.com/Shougo/neobundle.vim
