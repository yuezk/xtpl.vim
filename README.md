# xtpl.vim

XTemplate Syntax file for Vim

仿照 [vim-mustache-handlebars][handlebar] 这个项目，实现了对 XTemplate 的语法高亮。

## 如何安装

使用 [Vundle][vundle]:

```
Plugin 'yuezk/xtpl.vim'
```

使用 [vim-plug][plug]:

```
Plug 'yuezk/xtpl.vim'
```

使用 [NeoBundle][bundle]:

```
NeoBundle 'yuezk/xtpl.vim'
```
